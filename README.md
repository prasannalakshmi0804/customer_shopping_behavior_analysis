# Customer Shopping Behaviour Analysis

An end-to-end data analytics project using **Python**, **MySQL**, and **Power BI**.

## Tools Used

- **Python** — Data cleaning and transformation
- **MySQL** — Business analysis using SQL queries
- **Power BI** — Interactive dashboard and visualization

## Project Files

- `CUSTOMER_BEHAVIOUR_.ipynb` — Data cleaning and transformation
- `customer_shopping_slqquery.sql` — 11 business SQL queries
- `CUSTOMER_SHOPPING_ANALYSIS.pbix` — Power BI dashboard
- `customer_shopping_behavior.csv` — Raw dataset

## How to Run

1. Place `customer_shopping_behavior.csv` in the same folder as the notebook
2. Set your database password: `set DB_PASSWORD=yourpassword`
3. Run the notebook — it will clean the data and load it into MySQL
4. Open the SQL file in MySQL Workbench and run the queries
5. Open the `.pbix` file in Power BI Desktop
